# ✊✋✌️ Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors Game** built with **HTML, CSS, and JavaScript**. Challenge the computer, make your move, and see who wins!

---

# 📌 Project Overview

This project is a browser-based implementation of the classic **Rock Paper Scissors** game. It was created to practice fundamental JavaScript concepts such as event handling, conditional logic, random number generation, and DOM manipulation.

The player chooses **Rock**, **Paper**, or **Scissors**, while the computer randomly selects its move. The game compares both choices and instantly displays the result.

---

# ✨ Features

* 🎮 Play against the computer
* 🎲 Random computer move generation
* ⚡ Instant win, lose, or draw result
* 📢 Displays both player and computer choices
* 🔄 Play unlimited rounds
* 📱 Responsive and beginner-friendly interface

---

# 🛠️ Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Game logic and interactivity

# 🎯 Game Rules

* 🪨 Rock beats Scissors
* ✂️ Scissors beats Paper
* 📄 Paper beats Rock
* If both choices are the same, the game ends in a Draw.
